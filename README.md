## Featured Projects

### AgentFlow-AI — Agentic AI Research Assistant
> *Feb 2026*

An autonomous research assistant that orchestrates multi-step workflows using **LangGraph**, powered by **Groq LLM** for fast inference and **Tavily** for real-time web search. Built with a **Next.js** frontend and **FastAPI** backend, deployed on Vercel.

- Breaks down complex research queries into intelligent sub-tasks
- Synthesizes results into coherent, actionable insights
- Full-stack architecture with real-time AI-powered responses

### HealthGPT — Medical Chatbot
> *May 2025*

A **Streamlit**-powered medical Q&A chatbot using **LangChain**, **Mistral-7B-Instruct** via Hugging Face Inference API, and **FAISS** for semantic vector search over custom medical document embeddings.

- Conversational Q&A grounded in domain-specific documents
- Containerized with **Docker Compose** for portable deployment
- Clean chat UI with real-time LLM inference

[![View Project](https://img.shields.io/badge/View%20Project-181717?style=flat-square&logo=github)](https://github.com/Malyavantham-Radha-Krishna-Deekshith)

---

### MediInsight — Multi-Disease Prediction Web App
> *Apr 2024 · Published at ICSIE 2025*

A **Flask** web app predicting 7 diseases — Diabetes, Heart Disease, Kidney Disease, Liver Disease, Breast Cancer, Malaria, and Pneumonia — using both tabular ML models and CNN-based image classifiers (Keras/TensorFlow).

- Dual input: form-based data entry and medical image uploads (X-rays, cell images)
- Dockerized for seamless deployment and scalability
- Published: *"MediInsight: A HealthGPT-Powered Platform for Disease Diagnosis and Personalized Recommendations"* — 15th International Conference on Science and Innovative Engineering, April 2025

[![View Project](https://img.shields.io/badge/View%20Project-181717?style=flat-square&logo=github)](https://github.com/Malyavantham-Radha-Krishna-Deekshith)
